Setting up a passkey on the online account that supports them is easy and adds extra security to your account. Here’s how you can set it up:

  1. Go to your account’s security settings
    Log in to the website or app you want to secure (like Google, PayPal, or another supported platform). Find the Security or Account Settings section.

  2. Choose to create a passkey
    Look for Create a passkey, Set up passkey, or a similar option.

  3. Select the Passkeys App
    When asked where to save your passkey, choose Passkeys App by 2Stable and tap Continue.

  4. Confirm it’s you
    Authenticate using Face ID, Touch ID, or the Passkeys App password you’ve set up.

That’s it! Your passkey will be stored securely in the Passkeys App and ready to use next time you log in.

ℹ️ Tip: Passkeys are supported on an increasing number of platforms. If you don’t see the option yet on a service you use, check back later or follow their latest security updates.
