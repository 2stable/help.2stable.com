If you’ve saved passwords, documents, or 2FA codes in multiple vaults, you can consolidate them into one Primary Vault using the Export & Import features. This is especially useful if you’ve accidentally created data in different vaults or want to keep everything in one place for easier access and backup.

➡️ Step 1: Export Data from the Other Vault

  1. Open the Password Manager on the device where the data is currently stored.

  2. Go to Settings > General > Export.

  3. Read and confirm the terms, then tap Continue to generate an export file.

🔐 Note: The exported file is not encrypted. Anyone with access to it can view your data. Please store it securely and delete it once you’re done.

⬅️ Step 2: Import into Your Primary Vault

  1. Open the Password Manager on the device where your Primary Vault is displayed.

  2. Go to Settings > General > Import.

  3. Choose the appropriate source and file type.

  4. Tap Select File, locate the file, and import it.

Once the file is imported, your data should appear on the Password Manager main screen.
