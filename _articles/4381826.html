Passkeys offer a secure, passwordless way to sign in to supported services using Face ID, Touch ID, or your Password Manager password. With our Password Manager, saving and using passkeys is fast and secure. Here’s how to set it up:

  1. Go to your account’s security settings
    Open the website or app you want to secure (like Google, PayPal, or another supported service). Log in and head to the Security or Account Settings section.

  2. Choose to set up a passkey
    Look for an option like Create a passkey or Add a new sign-in method. Select Passkey.

  3. Select the Password Manager
    When your device asks where to save the passkey, choose Password Manager by 2Stable and tap Continue.

  4. Authenticate
    Use Face ID, Touch ID, or enter your Password Manager password to confirm it’s you.

  5. Choose or create the item

    • If the account already exists in your Password Manager, select it from the list.

    • If not, tap Create new item to save it as a new login.

  6. Save the passkey
    Once the passkey is added:

    • If you’re updating an existing item, scroll down to confirm it appears under Passkeys, then tap Save in the top right.

    • If it’s a new item, enter any additional info you’d like to store and then tap Save.

That’s it! The next time you sign in to that service, you’ll be able to use your passkey stored in the Password Manager, no need to remember a password.

ℹ️ Tip: Passkeys are supported on an increasing number of platforms. If you don’t see the option yet on a service you use, check back later or follow their latest security updates.
