The Password Manager allows you to back up and restore your passwords, documents, and codes by creating a file. This feature makes it easier to move your data between devices with different Apple IDs associated with them, or from other password manager apps.

To create a backup file of your saved data, follow these steps:

  1. Open the Password Manager on your device.

  2. Go to Settings > General > Export.

  3. Read and confirm the terms, and tap Continue.

üîê Note for security: The exported file is not encrypted, and anyone with access to it can view your data. Be sure to store it in a safe and private location.

‚¨ÖÔ∏è Importing a Backup
You can import a file created by our app or another password manager by following these steps:

  1. Open the Password Manager, then go to Settings > General > Import.

  2. In the Import section, select and tap the source and file type.

  3. Tap Select File, locate and select the file you wish to import to complete the process..

Once the file is successfully imported, your data should appear on the main screen of the Password Manager.
