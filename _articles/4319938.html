The Authenticator App allows you to back up and restore your 2FA codes using a .json file. This feature makes it easier to move your data between devices with different Apple IDs associated with them, or from other authenticator apps.

Exporting Your Accounts
To create a backup file of your current 2FA accounts:

  1. Open the Authenticator App on your device.

  2. Go to Settings > Import / Export.

  3. Under the Export section, tap Create Backup File.

üîê Note for security: The exported file is not encrypted, and anyone with access to it can view your 2FA account data. Be sure to store it in a safe and private location.

Importing a Backup
You can import a .json file created by our app or another authenticator that supports the same format:

  1. Open the Authenticator App, then go to Settings > Import / Export.

  2. In the Import section, tap Select Backup File.

  3. Locate and select your .json file, then tap Import to complete the process.

Once the file is successfully imported, your 2FA codes will appear on the main screen of the Authenticator App.
